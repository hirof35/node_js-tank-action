# 🚀 2D マルチプレッサー戦車バトルゲーム

Node.js, Socket.io, および HTML5 Canvas (JavaScript) を使用して開発した、リアルタイム2D対戦戦車ゲームです。

## 🎮 ゲームの特徴
- **リアルタイム同期:** WebSockets (Socket.io) を用いた、複数プレイヤーによる同画面での撃ち合い。
- **武器切り替えシステム:** 威力の高い「主砲」と、連射性能に優れた「機銃」をQキーで切り替え可能。
- **インテリジェントBot AI:** プレイヤーを自動で追跡・攻撃するNPC敵戦車を搭載。
- **リアルタイム難易度変更:** 画面上部のボタンから、Botの強さを「かんたん」「ふつう」「むずかしい」に即座に変更可能。
- **戦術的要素:** 障害物（壁）による射線の遮断、および定期的にスポーンする「回復アイテム」の争奪戦。
- **ジュース（演出）のこだわり:** 弾が壁や敵に接触した際、および撃破時にド派手なパーティクル（爆発火花）が散るアニメーションを実装。

## 🕹️ 操作方法
- **移動 / 旋回:** 矢印キー（`↑``↓``←``→`）
- **攻撃 (発射):** スペースキー（機銃は押しっぱなしで連射可能）
- **武器切り替え:** `Q` キー
